# Hubbett Control (Deutsch) – Unsigned iOS Projekt (iOS 17+)
Dieses Repository ist für Codemagic vorbereitet und erzeugt eine installierbare `.ipa` ohne Apple-Entwicklerkonto.

## Build
- Workflow: `ios-app`
- Artifact: `build/ipa/HubbettControl.ipa`

## Features
- Dark Mode, Hold-to-Run Buttons, Auto‑Waage, Stopp.
- BLE-Suche nach Namen, die „Hubbett“ enthalten.
- Neigung (Pitch/Roll) per CoreMotion.
